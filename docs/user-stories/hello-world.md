# Hello World

This is a sample user story page. Use it as a template for writing your own.

## What I wanted to do

I wanted to test whether dropping a markdown file into the `user-stories` folder would automatically show up on the documentation site.

## What happened

It worked! The build script picked up the page, added it to the navigation, and rendered it alongside everything else.

## Takeaway

Adding new pages is as simple as creating a `.md` file with frontmatter in the `docs/user-stories/` folder and rebuilding the site.
