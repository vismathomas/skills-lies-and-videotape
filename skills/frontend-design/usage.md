# Frontend Design

Creates distinctive, production-grade frontend interfaces with high design quality. Avoids generic 'AI slop' aesthetics by applying real design principles — typography, color theory, spacing, and visual hierarchy.

## Usage Examples

### Design a landing page

```
Create a visually distinctive landing page for a developer tool — no generic blue gradients.
```

### Build a dashboard UI

```
Design a monitoring dashboard with dark theme, data cards, and real-time status indicators.
```

### Create a component library

```
Build a set of form components with consistent styling and good accessibility.
```
