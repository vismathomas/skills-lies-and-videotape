# Agent Browser (CLI)

Declarative browser automation using the Vercel agent-browser CLI. Uses ref-based interactions for fast, exploratory browser tasks without generating full scripts.

## Usage Examples

### Quick website check

```
Open https://example.com and take a screenshot.
```

### Fill out a form

```
Navigate to the signup page, fill in the email field, and click submit.
```

### Capture evidence of a UI issue

```
Open the dashboard, take a screenshot, then click the settings button and take another screenshot.
```
