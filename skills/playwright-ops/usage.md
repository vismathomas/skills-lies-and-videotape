# Playwright Ops (Reusable Browser Operations)

Builds reusable, composable browser operations for admin workflows, evidence capture, and runbook automation. Supports both CLI macros for quick execution and code adapters for maintainability.

## Usage Examples

### Create an admin operation

```
Define a reusable browser operation for creating users in the admin panel.
```

### Evidence capture workflow

```
Set up a Playwright operation that captures screenshots and traces for compliance auditing.
```

### Automate a runbook step

```
Convert this manual runbook step into a deterministic browser operation.
```
