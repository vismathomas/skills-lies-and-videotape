# Gherkin + Playwright Integration

Integrates Gherkin/BDD feature files with Playwright browser automation. Covers locator strategy, page objects, auth reuse, and eventual consistency patterns. Multi-language: TypeScript, Python, C#.

## Usage Examples

### Connect features to Playwright

```
Wire up our login.feature Gherkin file to Playwright step implementations in TypeScript.
```

### Set up auth state reuse

```
Configure Playwright to reuse auth state across our BDD scenarios.
```

### Handle eventual consistency

```
Our event-sourced backend has propagation delays — set up Playwright assertions with polling.
```
