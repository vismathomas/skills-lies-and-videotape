# Markdown to Jira Conversion

Converts standard markdown syntax into Jira-compatible wiki markup. Handles headings, bold/italic, code blocks, links, tables, nested lists, and other formatting differences.

## Usage Examples

### Convert a README

```
Convert this README.md to Jira markup so I can paste it into a Jira issue.
```

### Convert release notes

```
Convert CHANGELOG.md to Jira format for the release issue.
```

### Quick conversion

```
Convert this markdown text to Jira markup:\n\n## Features\n- **Bold item**\n- `code item`
```
