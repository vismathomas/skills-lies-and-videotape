# Article Verification

Systematically deconstructs written content into verifiable claims, validates each using search and documentation, and facilitates informed discussion through structured interviewing. Ideal for fact-checking articles before publication.

## Usage Examples

### Fact-check a blog post

```
Verify all claims in this blog post about React performance optimizations.
```

### Validate a technical article

```
Check the factual accuracy of this article about Kubernetes scaling strategies.
```

### Review a press release

```
Identify and verify every factual claim in this product announcement.
```
