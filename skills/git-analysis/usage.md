# Git Repository Analysis

Analyzes git repositories for insights including contributor statistics, commit patterns, branch health, and change analysis. Produces actionable reports from repository history.

## Usage Examples

### Generate contributor stats

```
Show contributor statistics for the last 3 months.
```

### Analyze commit patterns

```
What are the commit frequency patterns and hotspots in this repo?
```

### Branch health check

```
Which branches are stale and which have unmerged work?
```
